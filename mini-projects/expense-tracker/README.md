# Personal Expense Tracker (CLI)

A beginner friendly Python command-line application that helps users manage their monthly budget by tracking expenses, viewing remaining balance, and receiving overspending warnings.

## Features
- Set a monthly budget
- Add expenses dynamically
- View remaining balance
- Overspending warning when balance goes below zero
- Simple and clear command-line interface

## Skills & Concepts Used
- Python functions
- `while` loops
- Conditional statements (`if / elif / else`)
- User input handling
- State management with variables
- Basic financial logic
- Currency formatting (PLN / z≈Ç)
