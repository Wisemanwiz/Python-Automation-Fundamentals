# Chapter 2 — Flow Control (Decision Making)

This chapter focuses on learning how to control the flow of a Python program using
conditional statements and logical comparisons.

## Topics Covered
- Boolean values (True and False)
- Comparison operators
- if, elif, else statements
- User input validation
- Decision making in programs

## Programs Implemented

### 1. Password Checker
A program that asks the user for a password and checks if it matches the stored password.

### 2. Even or Odd Checker
A program that asks the user for a number and determines whether it is even or odd.

### 3. Grading System
A program that asks the user for a score (0–100) and prints the corresponding grade:
- A (90+)
- B (80–89)
- C (70–79)
- D (60–69)
- F (below 60)

### 4. Login System
A simple login simulation that checks a username and password combination.

### 5. Number Guessing Game
A program that stores a secret number and asks the user to guess it.
